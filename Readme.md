Phoneme-level alignments for the CoVox dataset using the CatIPA phonetic standard.

This repository provides manually curated labels (e.g., .lab files) for selected CoVox recordings, designed for singing voice synthesis and phonetic research.

All phoneme annotations follow the CatIPA mapping to ensure consistency and compatibility with modern TTS and singing voice models.

⚠️ Note: This repository contains only labels. Audio data must be obtained separately from the original CoVox dataset.
